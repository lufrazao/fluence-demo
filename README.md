# Fluence Product Demo

Interactive product demo for Fluence, a behavioral intelligence API for fintech.

## Tech Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Recharts
- Lucide React

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Project Structure

- `src/components`: Reusable UI components
- `src/data`: Mock data and API simulation
- `src/pages`: Main application pages
- `src/hooks`: Custom React hooks
- `src/styles`: Global styles and Tailwind configuration

## Features

- **Dashboard**: High-level metrics and archetype distribution
- **Customer Intelligence**: Detailed profiles with behavioral signals and predictions
- **Next Best Action**: AI-driven recommendations engine
- **API Explorer**: Interactive playground for developer experience
- **Case Study**: Before/After impact visualization

## Deployment

This project is ready to be deployed to Vercel, Netlify, or any static hosting service.
